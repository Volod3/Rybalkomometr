# 🎣 Рибалкометр v1.0

ESP32-пристрій який аналізує погоду, тиск, фазу місяця та сезон — і видає прогноз кльову на LCD дисплей з RGB індикатором.

Базується на [AlexGyver MeteoClock v1.5](https://github.com/AlexGyver/MeteoClock), адаптовано під ESP32 + власна розпіновка.

---

## Можливості

- 📊 Графіки температури, вологості та тиску (погодинний і денний)
- 🐟 Прогноз кльову з урахуванням:
  - атмосферного тиску (локальний датчик + адаптація риби до змін)
  - фази місяця
  - сезону та нересту
  - часу доби
  - типу риби (мирна / хижак / карась / короп)
  - типу водойми (озеро/ставок або річка)
- 🌈 RGB LED: колір = рівень кльову (веселка = клює відмінно!)
- ☀️ Схід/захід сонця з інтернету (open-meteo.com, без API ключа)
- 🌐 WiFi: NTP синхронізація, прогноз погоди, веб-інтерфейс налаштувань
- 📡 Власна точка доступту (AP) для первинного налаштування
- 💡 Автояскравість дисплея та LED через фоторезистор
- 🕐 RTC DS3231 — час зберігається без WiFi
- 💾 Налаштування зберігаються в NVS (переживають перепрошивку)

## Залежності (PlatformIO / Arduino)

| Бібліотека | Призначення |
|---|---|
| `LiquidCrystal_I2C` | LCD 20×4 I2C |
| `RTClib` | RTC DS3231 |
| `Adafruit BME280` | Датчик температури/вологості/тиску |
| `ArduinoJson` | Парсинг відповідей open-meteo API |
| `WiFi`, `WebServer`, `HTTPClient` | Вбудовані в ESP32 Arduino core |
| `Preferences` | Збереження налаштувань у NVS |

## Схема підключення

| Компонент | Пін ESP32 |
|---|---|
| LCD SDA | GPIO 21 |
| LCD SCL | GPIO 22 |
| LCD підсвітка (BL) | GPIO 15 |
| BME280 SDA | GPIO 21 |
| BME280 SCL | GPIO 22 |
| RTC DS3231 SDA | GPIO 21 |
| RTC DS3231 SCL | GPIO 22 |
| RGB LED — R | GPIO 13 |
| RGB LED — G | GPIO 12 |
| RGB LED — B | GPIO 14 |
| Фоторезистор (LDR) | GPIO 34 (ADC1) |
| Кнопка TTP223 | GPIO 4 |

> BME280 адреса: `0x76` або `0x77` — визначається автоматично.  
> LCD адреса: `0x27` (змінюється в `#define DISPLAY_ADDR`).

## Режими дисплея (перемикання кнопкою)

| Режим | Опис |
|---|---|
| 0 | Годинник + поточні дані датчиків |
| 1 | Графік температури за день |
| 2 | Графік вологості за день |
| 3 | Графік тиску за годину |
| 4 | Графік тиску за день |
| 5 | Екран WiFi (SSID, пароль, IP) |
| 6 | Графік кльову за день |

## RGB індикатор кльову

| Колір | Рівень |
|---|---|
| 🔴 Червоний | Погано (Bad) |
| 🟡 Жовтий | 50/50 |
| 🟢 Зелений | Добре (Good) |
| 🌈 Веселка | Відмінно (Grt) — клює! |

## WiFi та налаштування

При першому старті (або довгому натисканні кнопки) пристрій піднімає точку доступу:
- **SSID:** `RybalkAP`
- **Пароль:** `fish1234`
- **IP:** `192.168.4.1`

Через веб-інтерфейс можна задати:
- мережі WiFi (до 3 штук)
- тип риби та водойми
- базовий тиск місцевості (2 слоти A/B)
- координати для прогнозу (2 слоти A/B)
- часовий пояс
